Reverse Engineering Project

Directions: Identify which type of programmer(Front End, Back end, Mobile Programmer or Data Analyst) would work on the features of Project Compass. Type your answers between the paragraph tags and be sure to provide an brief explanation for your choice.

1. Which programmer would build and manage the database for storing map pin locations in Project Compass?

Back end programmer

Relational databases is one of the top priorities of backend programmers; in dynamic web pages, most of the content is produced on flight by the exploration of big data bases. While front end programmers specialize in how this information is displayed, back end programmers manage the server applications, as the databases, with which user does not interact directly, but are supporting all of his experience in the web navigation.

2. Which programmer would make Project Compass responsive on different screen sizes on the web?

Front end programmer

Front end programmers are responsible for the user experience while interacting with a website. Is their labor to give the user the best possible experience, no matter what his device is, so they use responsive web design as a tool to make the page accesible.

3. Which type of mobile programmer (Android or iOS) would be using Swift to turn Project Compass into a mobile app?

iOS programmer

Android uses Java (and perhaps C++); iOS uses swift and objective C.

4. Which programmer would add the image to Project Compass?

Frontend programmer

Images are part of how the web page is displayed, and this is responsability of the front end programmer. He also gives the structure to the content, so is his job to put the <img> element in the HTML document.

5. Who would evaluate information about Project Compass users to give insights for product decisions?

Data analyst

The visitors of the website will produce a lot of data; handling it, and getting useful information fromt there is responsibility of data analyst. The obtained information could serve the team as a way to take decisions in the future - data driven decisions.

6. Who would make requests to the Google Maps API in Project Compass?

Front end programmer

APIs are ways to make the applications talk to each other; in this case, a front end developer would display information on a map to the user, using the Google Maps API.

7. Which programmer would need to consider hardware components to make Project Compass usable with a microphone or camera?

Mobile programmer

In web applications, the browser (and the underlying layers, like the OS) handles all of the hardware components. In mobile development, hardware management becomes part of the labor of the programmer, which requires additional effort from the programmer, but also gives a lot of possibilities.

8. Which programmer would make a post request endpoint to add comments onto Project compass?

Back end programmer

The back end programmer is responsible for planning the response that the server gives to user requests. The back end programmer will plan the requests that the clients will make to the server while sending forms (like a comment on a website), and the response the server will give to the client. According to this planeation, he will specify to the front end programmer how to send the requests, through <form> elements, and action - name - type - id attributes.

9. Which programmer would look at user clicking rates on Project Compass to make recommendations for external tourism sites?

Data analyst

This is part of the data produced by the visitors, and requires some statisctical techniques to obtain trends and useful information.
